# Visual-Basic-Core-Compiler Tasks

## 使用说明

- [ ] 表示未完成
- [x] 表示已完成
- 一期目标是先做出结构完整、语义稳定、可测试、可输出目标代码的 Visual Basic 核心编译器
- 当前优先打通 `VB -> AST -> IR -> Portable C` 的主链路，再决定是否扩展原生后端
